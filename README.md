# Movie_Recommender
A machine learning project implementing collaborative and content-based filtering algorithms to recommend movies to users.
 Features

-> Collaborative Filtering: Recommends movies based on user behavior and preferences of similar users
-> Content-Based Filtering: Recommends movies based on movie features (genre, year)
-> Data Visualization: Interactive charts showing rating distributions and popular movies
-> Sample Dataset: Includes 15 popular movies with 400+ synthetic ratings from 50 users
